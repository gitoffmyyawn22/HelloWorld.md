## This is a markdown file

Syntax Cheatsheet:
Phrase Emphasis
*italic*   **bold**
_italic_   __bold__

Headers
Setext-style:
Header 1
========

Header 2
--------
atx-style (closing #'s are optional):
# Header 1 #

## Header 2 ##

###### Header 6
Lists
Ordered, without paragraphs:
1.  Foo
2.  Bar
Unordered, with paragraphs:
*   A list item.

    With multiple paragraphs.

*   Bar
You can nest them:
*   Abacus
    * answer
*   Bubbles
    1.  bunk
    2.  bupkis
        * BELITTLER
    3. burper
*   Cunning
